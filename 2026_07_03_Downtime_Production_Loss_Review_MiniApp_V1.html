How this tool works

This tool turns a department's weekly downtime log into an instant loss figure and efficiency review, without manual arithmetic.

1. Name the cycle

Enter the facility or department, then pick the week start and end dates from the calendar pickers. The week label fills in automatically.

2. Add downtime entries — the fast way

If you have several rows to enter, use “Paste several rows at once”. Paste lines in the order cause, unit, machines, minutes, date, and the tool builds all the entries for you in one step. This is the quickest way to enter a full week's log.

Adding entries one at a time

Use “+ Add one entry” for a single new row, or “Duplicate last entry” when the next entry is nearly identical to the one before it (same cause and unit, different minutes or date) — it copies the previous row so you only need to change what's different. Each entry's date uses a calendar picker and defaults to the week's start date to save re-typing it every time.

3. Enter the week's production figures

Add actual production and target production in kilograms. Efficiency is calculated automatically.

4. Read the summary

The summary shows total loss, efficiency, and which cause is responsible for the most loss this week. A leading-cause flag appears automatically once one category passes 40 percent of total loss.

5. Save and compare cycles

Use “Save this cycle” to keep a week's entries in this browser. Saved weeks appear as chips you can reopen at any time to compare week on week. “New cycle” clears the form to start the next week without deleting saved ones.

6. Share the result

Use “Print / save as PDF” for a fileable one-page copy, or “Copy summary as text” to paste the headline figures into an email or chat message.

Note on data

Entries are stored only in the browser you are using. They are not sent anywhere. If you switch computers or clear browser data, saved cycles will not carry over — print or copy anything you need to keep.
